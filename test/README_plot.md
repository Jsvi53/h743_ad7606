# AD7606数据波形绘制工具使用说明

## 功能
从串口输出的文本数据中提取AD7606采集的电压值，并绘制波形图，用于验证采集数据是否与原始信号一致。

## 使用方法

### 方法1：从文件读取数据
1. 将串口输出的数据保存到文本文件（如 `ad7606_data.txt`）
2. 运行脚本：
   ```bash
   python plot_ad7606_data.py ad7606_data.txt
   ```

### 方法2：从标准输入读取
1. 运行脚本：
   ```bash
   python plot_ad7606_data.py
   ```
2. 粘贴串口输出数据
3. 按 `Ctrl+D` (Linux/Mac) 或 `Ctrl+Z` (Windows) 结束输入

### 方法3：直接使用提供的示例数据
如果已有数据文件，直接运行：
```bash
python plot_ad7606_data.py
```

## 输出
- 生成波形图：`ad7606_waveform.png`
- 显示统计信息：最大值、最小值、峰峰值、RMS值、平均值

## 数据格式要求
脚本会自动识别以下格式的数据行：
```
Sample #1000: V1=-2199 (0xF769) = -0.671 V
Sample #2000: V1=-30 (0xFFE2) = -0.009 V [Rate: 11904 SPS]
```

## 依赖库
```bash
pip install numpy matplotlib
```

## 注意事项
- 如果数据中包含采样率信息（`[Rate: XXX SPS]`），脚本会自动使用该采样率计算时间轴
- 如果没有采样率信息，将使用采样序号作为X轴
- 图形会显示完整波形和局部放大视图，便于观察细节
